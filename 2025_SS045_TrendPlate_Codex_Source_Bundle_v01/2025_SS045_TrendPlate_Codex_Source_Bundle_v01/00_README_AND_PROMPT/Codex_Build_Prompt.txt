3. Dynamic plate
Create a dynamic 11x17 landscape plate on sheet 14_Dynamic_Plate.
Add dropdown menus for Plate Number and Well ID.
The dropdowns must stay synchronized: if the user changes Plate Number, update Well ID; if the user changes Well ID, update Plate Number. Use a simple Worksheet_Change macro for synchronization.
The selected well should populate helper ranges on 13_Plate_Data_Current.
All charts must reference fixed ranges on 13_Plate_Data_Current.
Do not chart directly from FILTER spill ranges or volatile dynamic arrays.

4. Charts must plot
Every chart must display data when a valid well is selected. Use fixed helper ranges with #N/A for non-plotted values. Charts must update when the dropdown selection changes. Test at least D-1 BH01MW02, D-9 Well #15, D-12 A39MW094, and D-16 A39L036DPR2.

5. Axis titles and chart formatting
Axis titles must be actual Excel axis titles, not text boxes inside the plot area. Axis titles must appear outside the plot area.
VOC chart:
X-axis title: Date Sampled
Y-axis title: Concentration (µg/L)
Use true logarithmic y-axis where appropriate. Show BCT-CSL reference lines for relevant COCs. Label the 2025 result.
Use open symbols or distinct formatting for nondetects if plotted. Do not connect nondetects as trend points unless the plotting convention explicitly says so.

Water-level / precipitation chart:
Use a secondary axis only if necessary. Keep it visually subordinate to the VOC chart.

6. Mann-Kendall and trend logic
Include Mann-Kendall only as a supporting metric. Do not force Mann-Kendall where the dataset is not valid.
Use this rule:
MK is evaluated only where there are at least four valid independent detected results for a compound/well combination.
If not valid, display: MK: Not evaluated; insufficient detected results.

Also calculate and display:
Recent direction
2025 vs. 2024 result
2025 vs. historical peak
Current BCT-CSL status
Recommended 2026 action
